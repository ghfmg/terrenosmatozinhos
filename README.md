# Terrenos Matozinhos

Site institucional desenvolvido para apresentação de terrenos urbanos localizados em Matozinhos/MG.

## Objetivo

Disponibilizar informações claras, fotografias, localização, documentação e formas de contato dos imóveis.

## Estrutura

- Página inicial
- Terreno Rua Maria da Glória
- Área Paciência
- Contato

## Tecnologias

- HTML5
- CSS3
- JavaScript
